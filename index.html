About Me

Hi, I’m Judge, and I’m a passionate Information Security Analyst!

I specialize in penetration testing, network defense, vulnerability assessments, and digital forensics—helping organizations strengthen their security posture and safeguard critical systems.

Through hands-on projects, labs, and real-world simulations, I’ve built a strong portfolio of cybersecurity work covering network infrastructure, Active Directory exploitation, Wi-Fi security, web application testing, and digital forensics investigations.

My experience studying Information Science with a concentration in Cybersecurity at the University of Pittsburgh, combined with extensive projects and lab work, has given me practical exposure to both offensive security and defensive security.

My work includes penetration testing projects where I used Nmap, Nessus, and Metasploit to identify and exploit vulnerabilities.

I’ve configured and secured Cisco networks (VLANs, OSPF, BGP, DHCP, NAT) and deployed firewalls and intrusion detection with Snort.

I’ve conducted forensic investigations using FTK Imager, Autopsy, and Wireshark to analyze digital evidence and detect threats.

I’ve worked with Windows and Linux environments, setting up Active Directory attacks (ZeroLogon, password hash extraction/cracking) and hardening systems against them.

With a broad foundation across penetration testing, network engineering, and digital forensics, I bring both technical skills and a problem-solving mindset to the cybersecurity field. My goal is to contribute to defending organizations from ever-evolving cyber threats while continuing to sharpen my skills through certifications, hands-on labs, and real-world security projects.

Exploiting ZeroLogon to Gain Domain Admin Access

Used the ZeroLogon vulnerability (CVE-2020-1472) to exploit a misconfigured domain controller and escalate privileges. Identified the target system using Nmap, reset the machine account password using a custom Python script, and retrieved NTLM hashes with Impacket tools. Gained command-line access via WMIPExec and exfiltrated a local flag, demonstrating full domain compromise.
